### 设置扫描头参数

> 发送： cmd   ctrl   77   len   保存标志   key   value   
> 应答： cmd   ctrl   00   77    key   value   (这里00表示成功，01表示失败)

例子：  
> 设置参数 Continuous(01, 00)，参数号为 F189  
> 发送：`F0 04 77 04 00 F1 89 00`  
> 响应：`F1 04 00 77 F1 89 00`  最后的00为设置参数，若设置01返回的是01   

> 设置参数 Timeout (参数号为88)，设置3s，即30 = 0x1E  
> 发送：`F0 04 77 03 00 88 1E`  
> 设置成功响应: `F1 04 00 77 88 1E`  
> 设置失败响应: `F1 04 00 77`     无后面的参数号或值 



### 获取扫描头参数

> 发送： cmd   ctrl   76   len  key    
> 应答： cmd   ctrl   00   76   len  key   value   (这里00表示成功，01表示失败)

例子:  
> 获取参数 Continuous(01, 00) 参数号为 F189   
> 发送：`F0 04 76 02 F1 89`  
> 响应：`F1 04 00 76 03 F1 89 00`

> 获取参数 Timeout （参数号为88）  
> 发送：`F0 04 76 01 88`  
> 响应：`F1 04 00 76 02 88 1E`



### 获取多个扫描头参数

> 发送： cmd   ctrl   76   len  key1 key2 ...   
> 应答： cmd   ctrl   00   76   len  key1 value1 key2 value2 ...

例子：  
> 获取参数 codeId(2d) 和 timeout(88)
> 发送：`A5 5A 00 0D F0 04 76 02 2D 88 28 0D 0A`
> 应答：`A5 5A 00 0E F1 04 00 2D 00 88 63 3D 0D 0A`
